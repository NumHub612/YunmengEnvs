## 🎉 什么是 YunmengEnvs？

**YunmengEnvs**，云梦环境，作为一个融合 CFD 和 AI 技术的环境流体力学模拟和耦合框架，希望能帮助理解专业概念、数值方法和现实数据；希望能提供一个标准化的平台来尝试新想法、新技术和新方案。
